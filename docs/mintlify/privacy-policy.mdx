## Overview

This privacy policy explains how Lunch Pail Labs handles data across our add-ons and products. We are committed to protecting your privacy and ensuring transparent data practices.

## General Data Practices

### Data Collection

We collect minimal data necessary for core functionality:

- **Basic Usage Data**: We track basic interaction data to ensure functionality
- **Service Data**: Information required to provide our services
- **Technical Data**: Basic technical information needed for service delivery

### Data Usage

We use collected data solely to:

- Deliver our services and products
- Ensure proper functionality
- Improve user experience
- Maintain and upgrade our services

### Data Storage & Security

- All data is stored securely using industry-standard practices
- No personal data is sold to third parties
- We implement appropriate security measures to protect your data

## Product-Specific Policies

### PailPal Chrome Extension

#### Data Collection

- **Website Content**: We fetch and display workflow data from our database
- **Basic Usage Data**: We track workflow copying and basic interactions

#### Data Storage

- Workflow data is stored in our database
- No personal user data is collected or stored

#### Third-Party Services

- Database services for workflow storage
- Chrome's extension APIs for functionality

### Bubble Add-ons

- We only access data necessary for add-on functionality
- No data is stored outside of your Bubble app unless explicitly configured
- Add-on settings are stored within your Bubble app's configuration

### Adalo Components

- Components only access data necessary for functionality
- Data handling follows Adalo's platform guidelines
- No external data storage unless explicitly configured
