PailPal is a Chrome extension that helps you build faster with **Lunch Pail add-ons.**

![PailPal Extension](https://res.cloudinary.com/dpfmaccxx/image/upload/v1740398261/pailpalcovergif_xqp0am.gif)

## Key Features

| Feature                  | Description                                        |
| ------------------------ | -------------------------------------------------- |
| One-Click Workflow Setup | Install pre-built workflows instantly              |
| Best Practices Built-In  | Workflows follow platform-specific recommendations |
| Saves Time & Effort      | Skip manual setup and get started in minutes       |

## Installation

<Steps>
  <Step title="Install the extension">
    PailPal is currently **in review** in the Chrome Web Store. If you'd like
    **early access to the beta version**, [send a
    note](mailto:help@lunchpaillabs.com).
  </Step>
  <Step title="Pin the extension">
    Click the puzzle piece icon in Chrome and **pin PailPal** for easy access.
    <video
      controls
      className="w-full aspect-video"
      src="https://res.cloudinary.com/dpfmaccxx/video/upload/v1740397651/kgpk4yduztsxeutamyre.mp4"
    ></video>
  </Step>
  <Step title="Use PailPal">
    Click the PailPal icon when you're inside an editor to install a workflow.
    <video
      controls
      className="w-full aspect-video"
      src="https://res.cloudinary.com/dpfmaccxx/video/upload/v1740397191/tuxqp1so0r2ai48rq1mf.mp4"
    ></video>
  </Step>
</Steps>
