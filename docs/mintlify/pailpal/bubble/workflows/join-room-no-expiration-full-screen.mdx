## What This Workflow Does

This button **creates and joins a Daily video call room in full-screen mode** with no expiration. It's a great starting point for using the Daily plugin.

In a live app, you'd likely save the room details in your database‚Äîbut this workflow provides the **core structure** to create and join a room in one click.
