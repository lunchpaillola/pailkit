## What’s the Bundle?

![Demo](https://res.cloudinary.com/dpfmaccxx/image/upload/v1731529214/video_plugin_taerrc.png)

The **Video, Audio & Live Streaming Bundle** is a complete toolkit for adding advanced media capabilities to your Bubble apps. This bundle integrates tools for video conferencing, live streaming, and customizable media players, empowering you to create interactive and media-rich applications.

Included in this bundle are:

| **Feature**                                                                 | **Description**                                                                         |
| --------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| **[Daily Video Conferencing Plugin](/bubble/daily/getting-started)**        | An advanced version of the free Daily plugin for managing video and audio calls.        |
| **[MUX Live Streaming](/bubble/mux/getting-started)**                       | Real-time live streaming integration with MUX.                                          |
| **[Customizable Video Player](/bubble/player-style-video/getting-started)** | Embed and customize video files, live streams, and YouTube videos directly in your app. |
| **[Tavus](/bubble/tavus/getting-started)**                                  | Empower users to engage in real-time video conversations with AI avatars.               |
| **[VAPI](/bubble/vapi/getting-started)**                                    | Create AI phone assistants that handle inbound and outbound phone calls.                |

## Using the bundle

Download the plugin from the [Bubble marketplace](https://bubble.io/plugin/video-audio--live-bundle-1727441795463x240182085842370560). All actions and elements in the bundle are prefixed with "Bundle" for easy identification alongside other plugins. This bundle includes all features available in the free versions of our plugins.

To learn more about each plugin in the bundle, see the following:

- [Daily Video Conferencing Plugin](/bubble/daily/getting-started)
- [MUX Live Streaming](/bubble/mux/getting-started)
- [Customizable Video Player](/bubble/player-style-video/getting-started)
- [Tavus Conversational Video Interface ](/bubble/tavus/getting-started)
- [VAPI phone agents](/bubble/vapi/getting-started)
