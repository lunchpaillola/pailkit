The [VAPI](https://vapi.ai) plugin is part of the [Video, Audio & Live Streaming Bundle](/bubble/video-audio-live-bundle/getting-started) and allows you to create AI voice assistants that can handle phone calls in your application.

![VAPI Plugin](https://res.cloudinary.com/dpfmaccxx/image/upload/v1738721187/vapi-cover_tqjd8u.gif)

## Installation

<Steps>
  <Step title="Install the plugin">
    Go to the [Bubble Plugin
    Marketplace](https://bubble.io/plugin/video-audio--live-bundle-1727441795463x240182085842370560)
    and install the plugin into your app.
  </Step>
  <Step title="Verify installation">
    After installation, verify that the plugin appears in your list of installed
    plugins.
  </Step>
  <Step title="Add your VAPI credentials">
    In your [VAPI account](https://dashboard.vapi.ai), get your API key from the
    dashboard. Save this key as you'll need it in the next step.
  </Step>
  <Step title="Configure the plugin in Bubble">
    In your Bubble app, go to **Plugins** and find the **Video, Audio & Live
    Bundle** plugin. Enter your VAPI API key into the **VAPI API Key**
    field.
  </Step>
</Steps>
