### Fields

| **Field**     | **Type** | **Description**                                                                                                      |
| ------------- | -------- | -------------------------------------------------------------------------------------------------------------------- |
| **message**   | Text     | The message to send to participants.                                                                                 |
| **recipient** | Text     | Specifies who should receive the message. The default is \*, which sends to all participants except the broadcaster. |
| **user id**   | Text     | Set this field to send the message to a specific participant by their User ID.                                       |

### Returned Values

This action returns no values.
