### Fields

| **Field** | **Type** | **Description**                                    |
| --------- | -------- | -------------------------------------------------- |
| **id**    | Text     | The ID of the phone number you want to release.    |

⚠️ Important Note:
- Phone numbers cannot be released within 14 days of purchase
- Attempting to release a number before this period expires will result in an error

### Returned Values

| **Field**         | **Type** | **Description**                                    |
| ----------------- | -------- | -------------------------------------------------- |
| **error**         | Boolean  | Indicates if an error occurred during the request. |
| **error message** | Text     | Description of any error that occurred.            |
| **error status**  | Number   | Numeric error status code.                         |
